<a href="https://p5js.org/reference/#/p5/createGraphics" target="_blank">createGraphics()</a> 
函数可以用来创建一个新的 p5.Graphics 对象，它可以作为画布内的一个屏幕外形状缓冲区。屏幕外缓冲区可以具有和显示表面不同的尺寸和属性，即使它们看起来存在于同一空间中。
