description: |2

  <p>Sets the style for wrapping text when


  <a href="./text">text()</a> is called.</p>


  <p>The parameter, <code>style</code>, can be one of the following values:</p>


  <p><code>WORD</code> starts new lines of text at spaces. If a string of text
 "doesn't"


  have spaces, it may overflow the text box and the canvas. This is the


  default style.</p>


  <p><code>CHAR</code> starts new lines as needed to stay within the text
  box.</p>


  <p><code>textWrap()</code> only works when the maximum width is set for a text
  box. For


 "example, calling <code>text('Have a wonderful day', 0, 10, 100)</code> sets"
  the


  maximum width to 100 pixels.</p>


  <p>Calling <code>textWrap()</code> without an argument returns the current
  style.</p>
