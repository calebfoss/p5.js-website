  <p>A class to describe an


  <a
  href="https://developer.mozilla.org/en-US/docs/Learn/HTML/Introduction_to_HTML/Getting_started"
  target="_blank">HTML element</a>.</p>


  <p>Sketches can use many elements. Common elements include the drawing canvas,


  buttons, sliders, webcam feeds, and so on.</p>


  <p>All elements share the methods of the <code>p5.Element</code> class.
 "They're created"


  with functions such as <a href="/reference/p5/createCanvas">createCanvas()</a>
  and


  <a href="/reference/p5/createButton">createButton()</a>.</p>
