description: |2

  <p>Sets the font used by the <a href="./text">text()</a> function.</p>


  <p>The first parameter, <code>font</code>, sets the font.
  <code>textFont()</code> recognizes either


  a <a href="./p5.Font">p5.Font</a> object or a string with the name of a


 "system font. For example, <code>'Courier New'</code>.</p>"


  <p>The second parameter, <code>size</code>, is optional. It sets the font size
  in pixels.


  This has the same effect as calling <a href="./textSize">textSize()</a>.</p>


  <p>Note: <code>WEBGL</code> mode only supports fonts loaded with


  <a href="./loadFont">loadFont()</a>.</p>
