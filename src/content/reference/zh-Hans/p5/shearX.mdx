  <p>按照角度参数指定的量围绕 X 轴错切形状。角度应该以当前的 angleMode 指定。对象始终围绕它们相对于原点的位置错切，正数会将对象顺时针错切。</p>
  <p>变换应用于之后发生的所有事情，并且对函数的连续调用会累积效果。例如，调用 shearX(PI/2)，然后再调用 shearX(PI/2) 等同于调用shearX(PI)。如果在 <a href="/reference/p5/draw">draw()</a> 中调用 <a href="/reference/p5/shearX">shearX()</a>，则当循环重新开始时，变换会被重置。</p>
  <p>从技术上讲，<a href="/reference/p5/shearX">shearX()</a> 将当前变换矩阵乘以一个旋转矩阵。可以通过 <a href="/reference/p5/push">push()</a> 和 <a href="/reference/p5/pop">pop()</a> 函数进一步控制此函数。</p>
