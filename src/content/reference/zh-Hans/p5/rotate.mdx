  <p>通过角度参数指定的量旋转形状。此函数考虑到 <a href="/reference/p5/angleMode">angleMode</a>，因此可以在 RADIANS 或 DEGREES 中输入角度。</p>

  <p>从技术上讲，<a href="/reference/p5/rotate">rotate()</a> 将当前变换矩阵乘以一个旋转矩阵。可以通过 <a href="/reference/p5/push">push()</a> 和 <a href="/reference/p5/pop">pop()</a> 进一步控制此函数。</p>
