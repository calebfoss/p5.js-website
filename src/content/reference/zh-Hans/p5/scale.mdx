description: >
  <p>缩放坐标系。</p>

  <p>默认情况下，形状以其原始比例绘制。一个宽度为 50 像素的矩形在 100 像素宽的画布上看起来占据了一半的宽度。 <code>scale()</code> 函数可以缩小或拉伸坐标系，使形状以不同的大小呈现。有两种方式可以使用参数来调用 <code>scale()</code>，以设置缩放因子。</p>

  <p>第一种调用 <code>scale()</code> 的方式使用数字来设置缩放量。第一个参数 <code>s</code> 设置每个轴的缩放量。例如，调用 <code>scale(2)</code> 将 x 轴、y 轴和 z 轴拉伸 2 倍。接下来的两个参数 <code>y</code> 和 <code>z</code> 是可选的。它们设置 y 轴和 z 轴的缩放量。例如，调用 <code>scale(2, 0.5, 1)</code> 将 x 轴拉伸 2 倍，y 轴缩小 0.5 倍，z 轴保持不变。</p>

  <p>第二种调用 <code>scale()</code> 的方式使用一个 <a href="/reference/p5/p5.Vector">p5.Vector</a> 对象来设置缩放因子。例如，调用 <code>scale(myVector)</code> 使用 <code>myVector</code> 的 x、y 和 z 分量来设置沿 x、y 和 z 轴的缩放量。这样做与调用 <code>scale(myVector.x, myVector.y, myVector.z)</code> 是一样的。</p>

  <p>默认情况下，转换会累积。例如，连续两次调用 <code>scale(1)</code> 与调用一次 <code>scale(2)</code> 效果相同。可以使用 <a href="/reference/p5/push">push()</a> 和 <a href="/reference/p5/pop">pop()</a> 函数将转换隔离到不同的绘图组中。</p>

  <p>注意：转换会在绘制循环的开始处重置。在 <a href="/reference/p5/draw">draw()</a> 函数中调用 <code>scale(2)</code> 不会导致形状持续增长。</p>
