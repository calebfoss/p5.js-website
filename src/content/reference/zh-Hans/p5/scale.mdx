  <p>通过扩展或收缩顶点来增加或减少形状的尺寸。对象始终从它们相对于坐标系的原点进行缩放。缩放值以十进制百分比指定。例如，函数调用 scale(2.0) 会将形状的尺寸增加 200%。</p>
  <p>变换应用于之后发生的所有事情，并且对函数的连续调用会累积效果。例如，调用 scale(2.0)，然后再调用 scale(1.5) 等同于调用 scale(3.0)。如果在 <a href="/reference/p5/draw">draw()</a> 中调用 <a href="/reference/p5/scale">scale()</a>，则当循环重新开始时，变换会被重置。</p>
  <p>仅在 WebGL 模式下，此函数与 z 参数才可以一起使用。通过 <a href="/reference/p5/push">push()</a> 和 <a href="/reference/p5/pop">pop()</a> 可以进一步控制此函数。</p>
