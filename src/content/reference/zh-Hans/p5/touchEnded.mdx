description: >
  <p>每当屏幕触摸结束时调用的函数。</p>

  <p>声明函数 <code>touchEnded()</code> 会设置一个代码块，在用户停止屏幕幕设备时自动运行：</p>

  <code>

  function touchEnded() {
    // Code to run.
  }

  </code>


  <p>当 p5.js 调用 <code>touchEnded()</code> 时，<a href="/reference/p5/touches">touches</a> 数组将被更新为最新的触摸点：</p>

  <code>

  function touchEnded() {
    // Paint over the background.
    background(200);

  <p>  // Mark each remaining touch point when the user stops
    // a touch.
    for (let touch of touches) {
      circle(touch.x, touch.y, 40);
    }
  }

  </p></code>

  <p>参数 event 是可选的。<code>touchEnded()</code> 将被传递一个 <a href="https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent" target="_blank">TouchEvent</a> 对象，该对象具有描述触摸事件的属性：</p>

  <code>

  function touchEnded(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>在屏幕设备上，如果未声明 <code>touchEnded()</code>，当用户的触摸结束时，<a href="/reference/p5/mouseReleased">mouseReleased()</a> 将运行。如果声明了 <code>touchEnded()</code>，那么当用户的触摸结束时将运行 <code>touchEnded()</code>，而 <a href="/reference/p5/mouseReleased">mouseReleased()</a> 不会运行。</p>

  <p>注意：<a href="/reference/p5/touchStarted">touchStarted()</a>、<code>touchEnded()</code> 和 <a href="/reference/p5/touchMoved">touchMoved()</a> 是相关的。当用户触摸屏幕设备时，<a href="/reference/p5/touchStarted">touchStarted()</a> 会立即运行。当用户结束触摸时，<code>touchEnded()</code> 会立即运行。当用户移动任何触摸点时，<a href="/reference/p5/touchMoved">touchMoved()</a> 会重复运行。</p>
