  <p>每次点击结束时都会调用 <a href="/reference/p5/touchEnded">touchEnded()</a> 函数。如果没有定义 <a href="/reference/p5/touchEnded">touchEnded()</a> 函数，如果定义了 <a href="/reference/p5/mouseReleased">mouseReleased()</a> 函数，就会调用此函数。<br><br>

  浏览器可能会对各种点击事件附加不同的默认行为。要阻止点击事件的任何默认行为，请在方法的末尾添加 "return false"。</br></br></p>

      fill(value);
      rect(25, 25, 50, 50);
      describe('50-by-50 black rect turns white with touch.');
    }
    function touchEnded() {

        value = 255;
