  <p>一个 <code>String</code> 常量，用于设置 <a href="/reference/p5/angleMode">angleMode()</a>。</p>

  <p>默认情况下，诸如 <a href="/reference/p5/rotate">rotate()</a> 和 <a href="/reference/p5/sin">sin()</a> 等函数期望以弧度为单位测量角度。
  调用 <code>angleMode(RADIANS)</code> 可以确保以弧度为单位测量。在 <a href="/reference/p5/angleMode">angleMode()</a> 已设置为 <a href="/reference/p5/DEGREES">DEGREES</a> 的情况下，这样做可以很有用。</p>

  <p>注意：<code>TWO_PI</code> 弧度等于 360˚。
