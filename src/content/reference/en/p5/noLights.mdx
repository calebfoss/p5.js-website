  <p>Removes all lights present in a sketch.</p>
  <p>All subsequent geometry is rendered without lighting (until a new
  light is created with a call to one of the lighting functions
  (<a href="./lights">lights()</a>,
  <a href="./ambientLight">ambientLight()</a>,
  <a href="./directionalLight">directionalLight()</a>,
  <a href="./pointLight">pointLight()</a>,
  <a href="./spotLight">spotLight()</a>).</p>

  Three white spheres. Each appears as a different
  color due to lighting.
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100, WEBGL);
      describe(
        'Three white spheres. Each appears as a different color due to lighting.'
      );
    }
    function draw() {
      background(200);
      noStroke();

      ambientLight(255, 0, 0);
      translate(-30, 0, 0);
      ambientMaterial(255);
      sphere(13);

      noLights();
      translate(30, 0, 0);
      ambientMaterial(255);
      sphere(13);

      ambientLight(0, 255, 0);
      translate(30, 0, 0);
      ambientMaterial(255);
      sphere(13);
    }
    </code>
