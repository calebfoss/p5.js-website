description: >
  <p>Enable or disable perspective for lines in the WebGL renderer.
   The behavior of <code>linePerspective()</code>  is associated with the type of camera projection being used.</p>
  <ul>

  <li>When using <code>perspective()</code>, which simulates realistic
  perspective, linePerspective
    is set to <code>true</code> by default. This means that lines will be affected by the current
    camera's perspective, resulting in a more natural appearance.</li>
  <li>When using <code>ortho()</code> or <code>frustum()</code>, which do not
  simulate realistic perspective,
   linePerspective is set to <code>false</code> by default. In this case, lines will have a uniform
   scale regardless of the camera's perspective, providing a more predictable and
   consistent appearance.</li>
  <li>You can override the default behavior by explicitly calling
  <code>linePerspective()</code> after
   using <code>perspective()</code>, <code>ortho()</code>, or <code>frustum()</code>. This allows you to customize the line
   perspective based on your specific requirements.</li>
  </ul>
