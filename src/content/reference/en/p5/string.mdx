  <p>A sequence of text characters.</p>

  <p>The <code>String</code> data type is helpful for working with text. For
  example, a

  string could contain a welcome message:</p>

  <code>

  // Use a string literal.

  text('Hello!', 10, 10);

  </code>


  <code>

  // Create a string variable.

  let message = 'Hello!';


  <p>// Use the string variable.

  text(message, 10, 10);

  </p></code>

  <p>The most common way to create strings is to use some form of quotations as

  follows:</p>

  <code>

  text("hi", 50, 50);

  </code>


  <code>

  text('hi', 50, 50);

  </code>


  <code>

  text(`hi`, 50, 50);

  </code>


  <p><code>"hi"</code>, <code>'hi'</code>, and <code>hi</code> are all string
  literals. A "literal" means a

  value was actually written, as in <code>text('hi', 50, 50)</code>. By
  contrast,

  <code>text(message, 50, 50)</code> uses the variable <code>message</code>, so
  it isn't a string

  literal.</p>

  <p>Single quotes <code>''</code> and double quotes <code>""</code> mean the
  same thing. It's nice to

  have the option for cases when a string contains one type of quote:</p>

  <code>

  text("What's up?", 50, 50);

  </code>


  <code>

  text('Air quotes make you look "cool."', 50, 50);

  </code>


  <p>Backticks ```` create template literals. Template literals have many uses.

  For example, they can contain both single and double quotes as needed:</p>

  <code>

  text(`"Don't you forget about me"`,  10, 10); 

  </code>


  <p>Template literals are helpful when strings are created from variables like

  so:</p>

  <code>

  let size = random(10, 20);

  circle(50, 50, size);


  <p>text(<code>The circle's diameter is ${size} pixels.</code>,  10, 10);

  </p></code>

  <p>The <code>size</code> variable's value will replace <code>${size}</code>
  when the string is

  created. <code>${}</code> is a placeholder for any value. That means an
  expression can

  be used, as in <code>${round(PI, 3)}</code>. All of the following are valid
  template

  literals:</p>

  <code>

  text(`Ï€ is about ${round(PI, 2)} pixels.`,  10, 10);

  text(`It's ${mouseX < width / 2} that I'm on the left half of the canvas.`, 
  10, 30);

  </code>


  <p>Template literals can include several variables:</p>

  <code>

  let x = random(0, 100);

  let y = random(0, 100);

  let size = random(10, 20);

  circle(x, y, size);


  <p>text(<code>The circle at (${x}, ${y}) has a diameter of ${size}
  pixels.</code>,  10, 10);

  </p></code>

  <p>Template literals are also helpful for creating multi-line text like
  so:</p>

  <code>

  let poem = `My sketch doesn't run;

  it waits for me patiently

  while bugs point the way.`;


  <p>text(poem, 10, 10);

  </p></code>
