description: >
  <p>Executes the code within <a href="./draw">draw()</a> one time. This

  function allows the program to update the display window only when necessary,

  for example when an event registered by <a
  href="./mousePressed">mousePressed()</a>

  or <a href="./keyPressed">keyPressed()</a> occurs.</p>

  <p>In structuring a program, it only makes sense to call <a
  href="./redraw">redraw()</a>

  within events such as <a href="./mousePressed">mousePressed()</a>. This

  is because <a href="./redraw">redraw()</a> does not run

  <a href="./draw">draw()</a> immediately (it only sets a flag that indicates

  an update is needed).</p>

  <p>The <a href="./redraw">redraw()</a> function does not work properly when

  called inside <a href="./draw">draw()</a>.To enable/disable animations,

  use <a href="./loop">loop()</a> and <a href="./noLoop">noLoop()</a>.</p>

  <p>In addition you can set the number of redraws per method call. Just

  add an integer as single parameter for the number of redraws.</p>
