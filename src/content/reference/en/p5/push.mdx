description: >
  <p>The <a href="./push">push()</a> function saves the current drawing style

  settings and transformations, while <a href="./pop">pop()</a> restores these

  settings. Note that these functions are always used together. They allow you
  to

  change the style and transformation settings and later return to what you had.

  When a new state is started with <a href="./push">push()</a>, it builds on

  the current style and transform information. The <a href="./push">push()</a>

  and <a href="./pop">pop()</a> functions can be embedded to provide more

  control. (See the second example for a demonstration.)</p>

  <p><a href="./push">push()</a> stores information related to the current
  transformation state

  and style settings controlled by the following functions:

  <a href="./fill">fill()</a>,

  <a href="./noFill">noFill()</a>,

  <a href="./noStroke">noStroke()</a>,

  <a href="./stroke">stroke()</a>,

  <a href="./tint">tint()</a>,

  <a href="./noTint">noTint()</a>,

  <a href="./strokeWeight">strokeWeight()</a>,

  <a href="./strokeCap">strokeCap()</a>,

  <a href="./strokeJoin">strokeJoin()</a>,

  <a href="./imageMode">imageMode()</a>,

  <a href="./rectMode">rectMode()</a>,

  <a href="./ellipseMode">ellipseMode()</a>,

  <a href="./colorMode">colorMode()</a>,

  <a href="./textAlign">textAlign()</a>,

  <a href="./textFont">textFont()</a>,

  <a href="./textSize">textSize()</a>,

  <a href="./textLeading">textLeading()</a>,

  <a href="./applyMatrix">applyMatrix()</a>,

  <a href="./resetMatrix">resetMatrix()</a>,

  <a href="./rotate">rotate()</a>,

  <a href="./scale">scale()</a>,

  <a href="./shearX">shearX()</a>,

  <a href="./shearY">shearY()</a>,

  <a href="./translate">translate()</a>,

  <a href="./noiseSeed">noiseSeed()</a>.</p>

  <p>In WEBGL mode additional style settings are stored. These are controlled by
  the

  following functions: <a href="./setCamera">setCamera()</a>,

  <a href="./ambientLight">ambientLight()</a>,

  <a href="./directionalLight">directionalLight()</a>,

  <a href="./pointLight">pointLight()</a>, <a href="./texture">texture()</a>,

  <a href="./specularMaterial">specularMaterial()</a>,

  <a href="./shininess">shininess()</a>,

  <a href="./normalMaterial">normalMaterial()</a>

  and <a href="./shader">shader()</a>.</p>
