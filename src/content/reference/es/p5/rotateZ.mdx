description: |
  <p>La variable del sistema rotationZ siempre contiene la rotación del
  dispositivo a lo largo del eje z. Si el <a href="/reference/p5/angleMode">
  angleMode()</a> del boceto está configurado en GRADOS, el valor será de 0 a 360. Si
  está configurado en RADIANES, el valor será de 0 a 2*PI.</p>
  <p>A diferencia de rotationX y rotationY, esta variable está disponible solo para dispositivos
  con una brújula incorporada.</p>
  <p>Nota: El orden en que se llaman las rotaciones es importante, es decir, si se usan
  juntas, deben llamarse en el orden Z-X-Y o podría haber
  un comportamiento inesperado.</p>
