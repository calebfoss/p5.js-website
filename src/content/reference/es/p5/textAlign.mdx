description: >
  <p>Establece la forma en que el texto está alineado cuando se llama a <a href="/reference/p5/text">text()</a>.</p>

  <p>De forma predeterminada, llamar a <code>text('hi', 10, 20)</code> coloca la esquina inferior izquierda del cuadro del texto en (10, 20).</p>

  <p>El primer parámetro, <code>horizAlign</code>, cambia la forma en que <a href="/reference/p5/text">text()</a> interpreta las coordenadas x. De forma predeterminada, la coordenada x establece el borde izquierdo del cuadro delimitador. <code>textAlign()</code> acepta los siguientes valores para <code>horizAlign</code>: <code>LEFT</code>, <code>CENTER</code>, o <code>RIGHT</code>.</p>

  <p>El segundo parámetro, <code>vertAlign</code>, es opcional. Cambia la forma en que <a href="/reference/p5/text">text()</a> interpreta las coordenadas y. De forma predeterminada, la coordenada y establece el borde inferior del cuadro delimitador. <code>textAlign()</code> acepta los siguientes valores para <code>vertAlign</code>: <code>TOP</code>, <code>BOTTOM</code>, <code>CENTER</code>, o <code>BASELINE</code>.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Draw a vertical line.

      textSize(16);
      textAlign(RIGHT);
      text('ABCD', 50, 30);

      textAlign(CENTER);
      text('EFGH', 50, 50);

      // Bottom line.
      textAlign(LEFT);
      text('IJKL', 50, 70);

      createCanvas(100, 100);

      background(200);

      // First line.
      line(0, 12, width, 12);
      textAlign(CENTER, TOP);
      text('TOP', 50, 12);

      // Second line.
      line(0, 37, width, 37);
      textAlign(CENTER, CENTER);
      text('CENTER', 50, 37);

      line(0, 62, width, 62);
      textAlign(CENTER, BASELINE);
      text('BASELINE', 50, 62);

      // Fourth line.
      line(0, 97, width, 97);
      textAlign(CENTER, BOTTOM);
      text('BOTTOM', 50, 97);

    }
