description: >
  <p>Rota el sistema de coordenadas alrededor del eje y- en modo WebGL.</p>

  <p>El parámetro, <code>angle</code>, es la cantidad a rotar. Por ejemplo,
  llamar a

  <code>rotateY(1)</code> rota el sistema de coordenadas alrededor del eje y- por 1
  radián.

  <code>rotateY()</code> interpreta los valores de ángulo utilizando el

  <a href="/reference/p5/angleMode">angleMode()</a> actual.</p>

  <p>Por defecto, las transformaciones se acumulan. Por ejemplo, llamar
  <code>rotateY(1)</code>

  dos veces tiene el mismo efecto que llamar <code>rotateY(2)</code> una vez. Las

  funciones <a href="/reference/p5/push">push()</a> y <a
  href="/reference/p5/pop">pop()</a>

  pueden ser utilizadas para aislar transformaciones dentro de grupos de dibujo distintos.</p>

  <p>Nota: Las transformaciones se reinician al inicio del ciclo de dibujo. Llamar

  <code>rotateY(1)</code> dentro de la función <a href="/reference/p5/draw">draw()</a>
  no hará que

  las formas giren.</p>
