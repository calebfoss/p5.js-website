description: >
  <p>배열은 여러 개의 데이터를 순서대로 보관합니다.</p>

  <p>배열은 관련 데이터를 저장하는 데 유용합니다. 배열은 어떤 유형의 데이터라도 포함할 수 있습니다. 
  
  예를 들어, 배열은 문자열로 된 누군가의 좋아하는 색상 목록을 포함할 수 있습니다. 

  배열은 다음과 같이 생성됩니다:</p>

  <code>

  let colors = ['deeppink', 'darkorchid', 'magenta'];

  </code>


  <p>배열에서 각 데이터 조각을 원소라고 합니다. 각 원소는 배열 내에서 인덱스라고 하는 주소를 가지고 있습니다.      
  변수 <code>colors</code> 는 <a href="/reference/p5/String">String>타입의 세 요소인 <code>'deeppink'</code>, <code>'darkorchid'</code> 그리고 <code>'magenta'</code>를 포함하는 배열을 참조합니다.
  
  배열은 0부터 시작하는 인덱스를 사용합니다. 이는 <code>'deeppink'</code>가 인덱스 0에, <code>'darkorchid'</code>가 인덱스 1에, <code>'magenta'</code>가 인덱스 2에 위치한다는 것을 의미합니다.
  
  배열의 요소는 다음과 같이 각 요소의 인덱스를 사용하여 접근할 수 있습니다:</p>

  <code>

  let zeroth = colors[0]; // 'deeppink'

  let first = colors[1]; // 'darkorchid'

  let second = colors[2]; // 'magenta'

  </code>


  <p><code>push()</code>라는 메소드 호출을 통해 배열의 끝에 원소를 추가합니다: </p>

  <code>

  colors.push('lavender');


  <p>let third = colors[3]; // 'lavender'

  </p></code>
