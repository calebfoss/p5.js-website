description: >
  <p>카메라의 위치와 방향을 설정합니다.</p>
  
  <p><code>myCamera.camera()</code>를 사용하면 물체를 다양한 각도에서 볼 수 있습니다.
  
  이 함수는 모두 선택적인 아홉 개의 매개변수를 가지고 있습니다.</p>
  
  <p>첫 세 개의 매개변수 <code>x</code>, <code>y</code>, <code>z</code>는 카메라의 위치 좌표입니다.
  
  예를 들어, <code>myCamera.camera(0, 0, 0)</code>를 호출하면 카메라가 원점인 <code>(0, 0, 0)</code>에 위치합니다.
  
  기본적으로 카메라는 <code>(0, 0, 800)</code>에 위치합니다.</p>
  
  <p>다음 세 개의 매개변수 <code>centerX</code>, <code>centerY</code>, <code>centerZ</code>는 카메라가 바라보는 지점의 좌표입니다.
  
  예를 들어, <code>myCamera.camera(0, 0, 0, 10, 20, 30)</code>을 호출하면 카메라가 원점인 <code>(0, 0, 0)</code>에 위치하고 <code>(10, 20, 30)</code>을 바라봅니다.
  
  기본적으로 카메라는 원점 <code>(0, 0, 0)</code>을 바라봅니다.</p>
  
  <p>마지막 세 개의 매개변수 <code>upX</code>, <code>upY</code>, <code>upZ</code>는 "로컬" 공간에서 "상향" 벡터의 구성 요소입니다.
  
  "상향" 벡터는 카메라의 y축을 정렬합니다.
  
  예를 들어, <code>myCamera.camera(0, 0, 0, 10, 20, 30, 0, -1, 0)</code>을 호출하면 카메라가 원점 <code>(0, 0, 0)</code>에 위치하고
  
  <code>(10, 20, 30)</code>을 바라보며 "상향" 벡터가 카메라를 거꾸로 들었을 때와 같은 <code>(0, -1, 0)</code>으로 설정됩니다.
  
  기본적으로 "상향" 벡터는 <code>(0, 1, 0)</code>으로 설정됩니다.</p>
