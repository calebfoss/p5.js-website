  <p>एक सपाट आकार के भीतर एक छेद बनाना शुरू करता है।</p>

  <p><code>beginContour()</code> और <a href="/reference/p5/endContour">endContour()</a> फ़ंक्शन फ्लैट कस्टम आकृतियों के भीतर नकारात्मक स्थान बनाने की अनुमति देते हैं। <code>beginContour()</code> नकारात्मक स्थान में शीर्ष जोड़ना शुरू करता है और <a href="/reference/p5/endContour">endContour()</a> उन्हें जोड़ना बंद कर देता है।

  <code>beginContour()</code> और <a href='/reference/p5/endContour'>endContour()</a> को <a href='/reference/p5/beginShape'>beginShape()</a> और <a href="/reference/p5/endShape">endShape()</a> के बीच कॉल किया जाना चाहिए ।</p>

  <p>रूपांतरण जैसे <a href='/reference/p5/translate'>translate()</a>, <a href='/reference/p5/rotate'>rotate()</a>, और <a href='/reference/p5/scale'>scale()</a> <code>beginContour()</code> और <a href='/reference/p5/endContour'>endContour()</a> के बीच काम नहीं करता है|

  <code>beginContour()</code> और <a href="/reference/p5/endContour">endContour()</a> के बीच अन्य आकृतियों का उपयोग करना भी संभव नहीं है, जैसे <a href='/reference/p5/ellipse'>ellipse()</a> या <a href='/reference/p5/rect'>rect()</ a>।</p>

  <p>ध्यान दें: नकारात्मक स्थान को परिभाषित करने वाले शीर्षों को बाहरी आकार से विपरीत दिशा में होना चाहिए। सबसे पहले, बाहरी आकृति के लिए दक्षिणावर्त क्रम में शीर्ष बनाएं। फिर, नकारात्मक स्थान के लिए वामावर्त क्रम में शीर्ष बनाएं।</p>

    translate(50, 50);

    beginShape();
    // Exterior part of shape, clockwise winding
    vertex(-40, -40);
    vertex(40, -40);
    vertex(40, 40);
    vertex(-40, 40);
    // Interior part of shape, counter-clockwise winding
    beginContour();
    vertex(-20, -20);
    vertex(-20, 20);
    vertex(20, 20);
    vertex(20, -20);
    endContour();
    endShape(CLOSE);
